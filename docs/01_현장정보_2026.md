## 2. 골프 라운딩 규칙 (확정)
- **평일(주중)**: 18홀 + **9홀 무료 서비스**
- **주말·일본공휴일**: 18홀 (9홀 무료 서비스 없음)
- **입국일**: 점심 후 9홀 또는 일몰 라운딩 (9홀 라운딩 운영 ~26.8월 / 26.9월~ 첫날 일몰 약 3·4홀)
- **9홀 유료 추가** 가능 (¥4,400/인)
- **EV카트 운영 없음** ← 기존 SaiZen의 EVカート(20대 캡 등) 규칙은 14hills에 해당 없음
- 우천·폭설·골프장 사정으로 라운드 불가 시 환불 없이 대체 프로그램(리조트↔대체장소 왕복 무상)
